# 💊 Personalized Drug Recommendation System

## 📌 Project Overview

The Personalized Drug Recommendation System is a machine learning-based application that suggests appropriate medicines based on user symptoms or medical conditions. The system analyzes input data and provides relevant drug recommendations to assist in decision-making.

This project demonstrates the use of AI in healthcare to improve efficiency, accuracy, and accessibility of medical suggestions.

---

## 🚀 Features

* 🧠 Machine Learning-based drug prediction
* 📝 User input for symptoms/conditions
* 💊 Recommends suitable medicines
* ⚡ Fast and efficient prediction system
* 🌐 Can be integrated with web applications

---

## 🛠️ Technologies Used

* Python
* Machine Learning (Scikit-learn / Pandas / NumPy)
* Flask (for web interface)
* HTML/CSS (frontend)

---

## ⚙️ Installation & Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/aakritii08/Personalized-drug-recommendation-system.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Personalized-drug-recommendation-system
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Train the model:

   ```bash
   python train.py
   ```

5. Run the application:

   ```bash
   python app.py
   ```

---

## 📊 How It Works

1. User inputs symptoms or disease-related data
2. Data is preprocessed and passed to the trained ML model
3. The model predicts the most suitable drug
4. Result is displayed to the user

---

## ⚠️ Note

The trained model file (`.pkl`) is not included in this repository due to GitHub file size limitations.
You can generate the model by running the training script (`train.py`).
